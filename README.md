# Todo App

Простой список дел.В приложении можно добавлять новые задачи менять их статус на выполнено и удалять кликом по крестику. Все данные сохраняются в облачной базе и обновляются

## Стек
* **Фронтенд:** React Native / Expo Web
* **Бэкенд:** Hasura Engine (GraphQL)
* **База данных:** PostgreSQL
* **Деплой:** Vercel
